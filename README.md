## 🔥 Featured Project

### [E-dues](https://e-dues.netlify.app) — University Dues Payment Platform

A full-stack payment platform with Flutterwave integration, automatic payment splitting, PDF receipt generation, and QR code verification.

**What I built:**
- Dual-portal architecture (student payment flow + admin dashboard)
- Flutterwave split payments with subaccounts
- Compact PDF receipt generation with PDFKit
- QR code verification system for receipt authenticity
- Email delivery with Nodemailer + Resend

**Stack:** React · Vite · Tailwind CSS · Node.js · Express · MongoDB · Flutterwave API · PDFKit
